วิธีการติดตั้งและเริ่มต้นการใช้งาน

  • เสียบอะแดปเตอร์จ่ายไฟและสาย Lan เข้ากับ Philips Hue Bridge รอจนไฟสถานะ 3 ดวงติดจนครบ
  • จากนั้นติดตั้ง Philips Hue Bridge ลงในแอป
  • - เมื่อแอปค้นหา Philips Hue Bridge จนพบ ให้กด Connect
  • - กดปุ่มกลมๆบน Philips Hue Bridge 1 ครั้ง หน้าแอปจะเด้งไปหน้าถัดไปเมื่อเชื่อมต่อสำเร็จ
  • หลังติดตั้งเรียบร้อยให้เพิ่มอุปกรณ์หลอดไฟลงในแอป
  • - กด Create new > Create new room หรือ Create new zone ที่ต้องการ
  • - เพิ่มอุปกรณ์หลอดไฟลงในแอป กดที่ Add ได้สูงสุด 50 อุปกรณ์
  • เริ่มต้นการใช้งานการตั้งค่าต่างๆได้เลย
  • ***Philips Hue Bridge เป็นตัวเชื่อมต่อและควบคุม ต้องใช้ร่วมกับหลอดไฟ Philips Hue ด้วย (จำหน่ายแยก ไม่มีแถมมาในชุด)
